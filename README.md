## How to run this project?
If you just downloaded this repository.
1) Open the terminal
2) Go into the map Portfolio
    ```shell
    cd .\Portfolio
    ```
3) Install npm
    ```shell
    npm install
    ```
4) And run the project
    ```shell
    npm run dev
    ```

If you already installed npm.
1) Open the terminal
2) Go into the map Portfolio
    ```shell
    cd .\Portfolio
    ```
3) And run the project
    ```shell
    npm run dev
    ```
