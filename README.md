## How to run this project?
If you just downloaded this repository.
1) Open the terminal
2) Go into the map Portfolio
    ```shell
    cd .\Portfolio
    ```
3) Install npm
    ```shell
    npm install
    ```
4) Install react-router-dom
    ```shell
    npm install react-router-dom
    ```
5) Install sass
    ```shell
    npm install sass
    ```
6) Install lucid-react
    ```shell
    npm install lucide-react
    ```
7) And run the project
    ```shell
    npm run dev
    ```

If you already installed everything before and you just want te run the project.
1) Open the terminal
2) Go into the map Portfolio
    ```shell
    cd .\Portfolio
    ```
3) And run the project
    ```shell
    npm run dev
    ```
